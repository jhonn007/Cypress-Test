# Cypress-Test

Cypress-Test is repo to practice with Cypress.

## Installation
```bash
cd /your/project/path
```
## Download the repo, you can use:

```bash
git clone https://github.com/jhonn007/Cypress-Test.git
```

## Initialize the project

```bash
npm install
```
## Open Cypress

```bash
npx cypress open
```
